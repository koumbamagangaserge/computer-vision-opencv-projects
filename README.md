This repository presents a series of practical projects developed using OpenCV, focusing on fundamental concepts in computer vision and image processing.

## Objectives

- Understand core image processing techniques
- Apply computer vision methods in practical scenarios
- Develop structured thinking in visual data analysis

## Projects Included

### 1. Image Blending and Transition
- Smooth transition between images using weighted blending
- Application of cv2.addWeighted()

### 2. Multi-Color Detection using HSV
- Detection of multiple colors (Red, Green, Blue)
- Use of HSV color space and masking techniques

### 3. Edge Detection (Canny Algorithm)
- Real-time edge detection
- Adjustable thresholds using trackbars

## Key Learning Outcomes

These projects allowed me to:
- Understand how images are processed at pixel level
- Explore how computer vision systems interpret visual information
- Build a foundation for more advanced AI-based vision systems
